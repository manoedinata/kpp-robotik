# Directory structure

```
kpp_robotik/
├── Graph.cpp       --> Graph (node/edge-related) functions
├── Graph.h         --> Graph header definition (OOP)
├── KPPRobotik.cpp  --> Main logic (Dijkstra for best routes) functions
├── KPPRobotik.h    --> Main logic header definition (OOP)
├── main.cpp        --> Main program, executing the main logic function
```
